# Фаза 3: Глибока інтеграція з ОС Windows та преміум UX

**Мета:** Перетворити додаток на повноцінну десктопну програму, що забезпечує безшовний та професійний досвід для користувачів Windows.

**Технологічний стек фази:**
- **`Electron API`:** Активне використання вбудованих модулів (`dialog`, `app`, `Notification`, `BrowserWindow`).
- **`electron-updater`:** Стандартне рішення для впровадження надійної системи автоматичних оновлень.

## 3.1. Робота з файлами та системою

- [x] **Нативні діалогові вікна:** Повністю замінити всі веб-діалоги на нативні виклики через **`dialog` API** з Electron.
- [x] **Асоціація з файлами `.json`:** Зареєструвати додаток як обробник для файлів `.json`.
- [x] **Список останніх файлів (Jump List):** Інтегрувати **`app.addRecentDocument()`**.

## 3.2. Інтеграція з інтерфейсом Windows

- [x] **Індикатор прогресу на панелі завдань:** Використовувати **`win.setProgressBar()`**.
- [x] **Нативні сповіщення:** Замінити веб-сповіщення на системні за допомогою **`Notification` API**.
- [x] **Збереження стану вікна:** Зберігати та відновлювати розміри та позицію вікна.

## 3.3. Поведінка додатку

- [x] **Блокування єдиного екземпляру (`Single Instance Lock`):** Використати **`app.requestSingleInstanceLock()`**.
- [x] **Система автоматичних оновлень:** Інтегрувати бібліотеку **`electron-updater`** для фонової перевірки, завантаження та встановлення оновлень.
