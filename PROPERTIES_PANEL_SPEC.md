# Специфікація: Панель "Властивості"

Цей документ описує вміст та поведінку панелі властивостей для кожного типу об'єкта в додатку "Малювальник". Він слугує єдиним джерелом правди для розробки та тестування.

---

## 1. Загальна поведінка

- **Нічого не вибрано:** Панель прихована або відображає повідомлення "Нічого не вибрано".
- **Вибрано один об'єкт:** Панель відображає повний набір властивостей для цього типу об'єкта.
- **Вибрано кілька об'єктів:**
    - Панель відображає лише ті властивості, що є **спільними** для всіх вибраних об'єктів.
    - Якщо значення спільної властивості однакове для всіх об'єктів, воно відображається.
    - Якщо значення відрізняються, поле вводу відображає плейсхолдер (напр., "Різні значення") або залишається порожнім.
    - Зміна властивості в цьому режимі застосовується до **всіх** вибраних об'єктів.

## 2. Структура властивостей за типом об'єкта

### Стіна (`Wall`)
- **Геометрія (тільки для читання):**
    - `X1`, `Y1`: Координати початкової точки.
    - `X2`, `Y2`: Координати кінцевої точки.
    - `Довжина`: Розрахована довжина стіни.
- **Загальні:**
    - `Тип стіни`: Випадаючий список (`Зовнішня`, `Внутрішня`, `Перестінок`).
    - `Товщина`: Числове поле, оновлюється при зміні типу, але може бути змінено вручну.
    - `Колір`: Вибір кольору.

### Двері (`Door`), Вікно (`Window`), Сходи (`StairsObject`), Символ (`SymbolObject`)
- **Геометрія (тільки для читання):**
    - `X`, `Y`: Координати центральної точки.
    - `Ширина`: Ширина габаритного прямокутника.
    - `Висота`: Висота габаритного прямокутника.
- **Трансформація:**
    - `Кут (°)`: Кут повороту об'єкта.

### Текст (`TextObject`)
- **Текст:**
    - `Стиль`: Випадаючий список з доступними текстовими стилями.
    - `Вміст`: Багаторядкове текстове поле для редагування тексту.
    - `Висота`: Числове поле для висоти тексту у світових одиницях.
    - `Колір`: Вибір кольору.

### Полілінія (`PolylineObject`)
- **Геометрія (тільки для читання):**
    - `Довжина`: Загальна довжина полілінії.
    - `К-сть вершин`: Кількість вершин.
- **Загальні:**
    - `Замкнута`: Прапорець (checkbox).
    - `Колір`: Вибір кольору.
    - `Товщина лінії`: Числове поле.

### Коло (`CircleObject`)
- **Геометрія:**
    - `X центра`, `Y центра`: Координати центру (тільки для читання).
    - `Радіус`: Числове поле для редагування радіуса.
- **Загальні:**
    - `Колір`: Вибір кольору.
    - `Товщина лінії`: Числове поле.

### Дуга (`ArcObject`)
- **Геометрія (тільки для читання):**
    - `X центра`, `Y центра`: Координати центру.
    - `Радіус`: Радіус дуги.
    - `Поч. кут`: Початковий кут.
    - `Кін. кут`: Кінцевий кут.
- **Загальні:**
    - `Колір`: Вибір кольору.
    - `Товщина лінії`: Числове поле.

### PDF-підложка (`PdfUnderlay`)
- **Геометрія (тільки для читання):**
    - `X`, `Y`: Координати центру.
    - `Ширина`: Ширина.
    - `Висота`: Висота.
- **Трансформація:**
    - `Кут (°)`: Кут повороту.
- **Вигляд:**
    - `Прозорість`: Повзунок (slider) від 0 до 100.

### Шлях евакуації (`EvacuationPath`, `EmergencyEvacuationPath`)
- **Геометрія (тільки для читання):**
    - `Довжина`: Загальна довжина.
    - `К-сть вершин`: Кількість вершин.
- **Загальні:**
    - `Ширина лінії`: Числове поле.
    - `Колір`: Вибір кольору.

### Штрихування (`HatchObject`)
- **Трансформація:**
    - `Кут (°)`: Кут повороту штрихування.
- **Шаблон:**
    - `Шаблон`: Випадаючий список (`Суцільний`, `Лінії`, `Сітка`).
    - `Масштаб`: Числове поле.
    - `Колір`: Вибір кольору.

### Група (`GroupObject`)
- Відображається як "Виділено декілька об'єктів". Специфічні властивості групи (окрім габаритів, що відображаються в секції "Геометрія") не редагуються. Панель поводиться так само, як при множинному виділенні, показуючи лише спільні для всіх об'єктів у групі властивості, якщо такі є.
